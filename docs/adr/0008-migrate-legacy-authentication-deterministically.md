# Migrate legacy authentication deterministically

Nacos Search migrates legacy authentication settings once into explicit strategies: fully blank credentials become ANONYMOUS, TOKEN becomes NACOS_PASSWORD, BASIC becomes HTTP_BASIC, and HYBRID becomes NACOS_PASSWORD when legacy token authentication was enabled or HTTP_BASIC when it was disabled. A complete migrated secret is staged into the credential slot for the profile's assigned access revision before that revision is published. A profile with only half of its credentials populated retains the mapped strategy but is marked `CredentialIncomplete` and surfaced as `ConfigurationRequired`; the upgrade summary reports the migration and removal of the HYBRID execution path, so subsequent failures require an explicit user choice instead of a silent fallback.
